imchoose

Choose a bounding box with mouse

Syntax

rec2 = imchoose()

Parameters

rec2 :

Seletected Bounding Box

Description

This function allows user to select a bounding box interactively using mouse.

Examples

S = imread(fullpath(getIPCVpath() + "/images/puffin.png"));
imshow(S);
rec2 = imchoose();
title(string(rec2));
