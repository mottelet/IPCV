im2col

Convert image into series of columns

Syntax

y = im2col(x,blk)

Parameters

x :

Source Image

blk :

Block size [m,n]

Description

This function is used to convert the image to columns of vector for faster processing as Scilab perform better with vectorization code.

Examples

A = testmatrix('mag',4);
B = im2col(A,[3 3]);
C = mean(B,1);
D = matrix(C,[4,4])';
