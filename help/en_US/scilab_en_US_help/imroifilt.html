imroifilt

Filtering of a selected region

Syntax

imout = imroifilt(imin,f,mask)

Parameters

imin :

Input Image

f :

Input filter

mask :

Mask for the ROI

imout :

Output image

Description

This function perform filtering over the region specified in mask.

Examples

S = imread(fullpath(getIPCVpath() + "/images/balloons_gray.png."));
mask = imroi(S);
h = fspecial('unsharp');
S2 = imroifilt(S,h,mask);
imshow(S2);
