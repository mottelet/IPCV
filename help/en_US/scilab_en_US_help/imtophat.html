imtophat

Image tophat

Syntax

imout = imtophat(imin,se)

Parameters

imin :

Input image

se :

Structure element

imout :

Output image

Description

This operation is the difference between an input image and its opening

Examples

S = imread(fullpath(getIPCVpath() + "/images/morpex.png"));
se = imcreatese('ellipse',7,7);
S2 = imtophat(S,se);
imshow(S2);
