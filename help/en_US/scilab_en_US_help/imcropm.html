imcropm

Crop an image using mouse selection

Syntax

y = imcropm(S)

Parameters

S :

Source Image

y :

Subimage

Description

This function allows user to select the region of interest interactively using mouse.

Examples

S = imread(fullpath(getIPCVpath() + "/images/puffin.png"));
J = imcropm(S);
figure(); imshow(J);
